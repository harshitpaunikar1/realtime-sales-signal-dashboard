# Real-Time Sales Signal Dashboard

> **Domain:** Marketing / AdTech

## Overview

Marketing teams lacked a live view of field activity to steer campaigns and budgets. Updates from sales reps arrived late, lived in multiple sheets, were hard to reconcile, slowing decisions and dampening conversion. Without timely visibility, high-intent leads went cold, spend was not redirected fast enough, territory opportunities were missed, raising acquisition costs and risking lost revenue.

## Approach

- Aligned on KPIs with sales and marketing leaders; mapped data owners, cadence, decision points
- Connected mobile sales inputs via secure APIs/webhooks; standardized payloads and identifiers across sources
- Built Python streaming layer to validate, deduplicate, enrich events before storage
- Modeled Tableau dashboard for real-time views: territory heatmaps, pipeline velocity, alert tiles
- Implemented data quality checks, timestamp drift monitoring, audit logs; ran UAT with power users
- Delivered weekly increments; documented playbooks and onboarded users with role-based views
